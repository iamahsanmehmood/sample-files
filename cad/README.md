# CAD & Engineering

Sample CAD and engineering drawing files.

| File | Format | Description |
|------|--------|-------------|
| sample.dxf | DXF | AutoCAD Drawing Exchange Format - rectangle with circle |
| sample.step | STEP | ISO 10303 STEP file - 3D geometry exchange format |
| sample.stp | STP | STEP format (alternate extension) |
| sample.iges | IGES | Initial Graphics Exchange Specification |
| sample.igs | IGS | IGES format (alternate extension) |
