# Email & Contacts

Sample email, contact, and calendar files.

| File | Format | Description |
|------|--------|-------------|
| sample.eml | EML | Email message with headers, body, and text attachment |
| sample.msg | MSG | Outlook message format (minimal) |
| sample.mbox | MBOX | Mailbox file with 2 messages |
| sample.vcf | VCF | vCard 3.0 contact card |
| sample.ics | ICS | iCalendar event |
