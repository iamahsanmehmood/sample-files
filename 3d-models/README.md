# 3D Models

Sample 3D model files. Most contain a simple cube or pyramid geometry.

| File | Format | Description |
|------|--------|-------------|
| sample.stl | STL | ASCII STL - cube with 12 triangles |
| sample.obj | OBJ | Wavefront OBJ cube with normals |
| sample.mtl | MTL | Material file for OBJ |
| sample.gltf | glTF | JSON-based 3D format with embedded data |
| sample.glb | GLB | Binary glTF format |
| sample.dae | DAE | Collada XML format |
| sample.ply | PLY | Stanford PLY with vertex colors |
| sample.off | OFF | Object File Format |
| sample.wrl | WRL | VRML 2.0 scene |
| sample.amf | AMF | Additive Manufacturing Format (pyramid) |
| sample.3mf | 3MF | 3D Manufacturing Format (cube) |

**Note:** FBX, STEP, IGES, 3DS, BREP, and 3DM formats are planned for future addition.
