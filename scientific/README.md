# Scientific Data

Sample scientific data files for testing scientific computing and data analysis tools.

| File | Format | Description |
|------|--------|-------------|
| sample.hdf5 | HDF5 | Hierarchical Data Format with temperature and pressure datasets |
| sample.fits | FITS | Flexible Image Transport System - 64x64 astronomical image |
| sample.nc | NetCDF | Network Common Data Form with climate-like time series data |
| sample.mat | MAT | MATLAB data file with sine wave and matrix data |

**Note:** DICOM (.dcm) and NIfTI (.nii) formats are planned for future addition.
These formats require specialized medical imaging tools to generate valid files.
