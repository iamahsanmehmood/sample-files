# Spreadsheets

Sample spreadsheet files with employee data across 10 rows and 6 columns.

| File | Format | Description |
|------|--------|-------------|
| sample.xlsx | XLSX | Excel spreadsheet with styled headers and data |
| sample.xls | XLS | Legacy Excel format |
| sample.ods | ODS | OpenDocument Spreadsheet |
| sample.csv | CSV | Comma-separated values |
| sample.tsv | TSV | Tab-separated values |

Data schema: ID, Name, Email, Department, Salary, Start Date
