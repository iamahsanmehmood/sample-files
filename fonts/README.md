# Fonts

Sample font files based on Google Fonts' Roboto (Apache 2.0 / OFL licensed).

| File | Format | Description |
|------|--------|-------------|
| sample.ttf | TTF | TrueType Font |
| sample.otf | OTF | OpenType Font |
| sample.woff | WOFF | Web Open Font Format |
| sample.woff2 | WOFF2 | Web Open Font Format 2 (compressed) |
| sample.eot | EOT | Embedded OpenType (legacy IE support) |

Source: [Google Fonts - Roboto](https://fonts.google.com/specimen/Roboto)
