# 📁 Sample Files Collection

> A curated collection of 200+ sample files across every major file format.
> Perfect for testing file viewers, converters, validators, and other file-processing tools.

## 🎯 Use Cases

- Testing file viewer applications (like [OpenFile.io](https://openfile.io))
- Validating file format parsers and converters
- QA testing for upload/download features
- Learning about different file formats
