# Audio

Sample audio files. Base audio is a 5-second synthesized chord progression.

| File | Format | Description |
|------|--------|-------------|
| sample.wav | WAV | Uncompressed PCM audio (44.1kHz, 16-bit) |
| sample.mp3 | MP3 | Compressed audio (converted from WAV) |
| sample.ogg | OGG | Vorbis compressed audio |
| sample.flac | FLAC | Lossless compressed audio |
| sample.m4a | M4A | AAC audio in MP4 container |
| sample.aac | AAC | Raw AAC audio |
| sample.mid | MIDI | MIDI sequence - C major scale melody |
| sample.midi | MIDI | Same MIDI file with .midi extension |

All audio is synthetically generated and royalty-free.
