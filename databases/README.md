# Databases

Sample SQLite database files with 2 tables and 12 rows each.

| File | Format | Description |
|------|--------|-------------|
| sample.sqlite | SQLite | SQLite database |
| sample.db | DB | SQLite database (.db extension) |
| sample.sqlite3 | SQLite3 | SQLite 3 database |

### Schema
- **users**: id, name, email, role, active
- **products**: id, name, price, category, stock
