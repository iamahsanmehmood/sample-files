# Executables

Minimal executable files for testing format detection and analysis.

| File | Format | Description |
|------|--------|-------------|
| sample.exe | EXE | Minimal PE32 executable (does nothing useful) |
| sample.dll | DLL | Minimal PE32 DLL |
| sample.elf | ELF | Minimal 64-bit ELF executable header (Linux) |
| sample.so | SO | Minimal ELF shared object / library (Linux) |

**Warning:** These are minimal files for format testing only. They contain no malicious code but also no useful functionality.
